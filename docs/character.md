# 角色系统

## 一、属性生成

每局开始时，系统使用**随机种子**生成四维基础属性：

| 属性 | 缩写 | 运行时映射               | 影响                                           |
| ---- | ---- | ------------------------ | ---------------------------------------------- |
| 力量 | STR  | attack = Max(STR - 10, 0) | 攻击加成，自动叠加到所有攻击类卡牌的伤害       |
| 体质 | CON  | HP = CON × 15 + 30       | 生命值                                         |
| 速度 | SPD  | speed = SPD              | 行动次数、先手判定、刺客系倍率                 |
| 法力 | MANA | mana = round(MANA × 1.5) | 每回合可用法力                                 |

**生成规则**：

- 总点数固定为 **40 点**，在四维中随机分配
- SPD 下限为 3（避免 0 速度锁死）
- 玩家在开局看到属性后，可以选择**接受**或**重摇一次**（每局仅一次免费重摇）

## 二、运行时属性

| 属性             | 初始值            | 战斗间规则                                       |
| ---------------- | ----------------- | ------------------------------------------------ |
| HP               | CON × 15 + 30     | **每场战斗后回满**                               |
| attack（攻击加成）| Max(STR - 10, 0)  | 自动叠加到攻击类卡牌；可被事件/遗物/汲取修改     |
| speed            | SPD               | 可被事件/遗物/霜蚀/汲取修改                      |
| mana             | round(MANA × 1.5) | 每场战斗重置为满，回合内每回合回复 1（不超上限） |
| armor            | 0                 | 每场战斗开始清零                                 |
| 霜蚀/灼烧/毒药   | 0                 | 每场战斗开始清零                                 |
| 诅咒卡（卡组中） | 无                | 持久存在；战斗中对手塞入的仅存于克隆卡组，战斗后自动丢弃；负面事件获得的保留至被移除 |
| 汲取效果         | 无                | 每场战斗开始清零（被偷的属性恢复）               |

---

## 三、HP 管理

- HP 在**每场战斗结束后自动回满**至 maxHp
- 原因：残影对手按满血计算，若玩家 HP 不回满则 PvP 天然不公平
- 资源压力改由**金币经济**和**卡组构筑选择**承担，而非 HP 消耗
- 事件/遗物中仍可提供战斗中回复（如吸血），但战斗间不再需要额外回血手段
