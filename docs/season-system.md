# 赛季系统

## 一、赛季周期

每个赛季持续 **4 周**。

---

## 二、赛季重置内容

| 重置         | 说明                                             |
| ------------ | ------------------------------------------------ |
| 排行榜       | 清零，重新竞争                                   |
| 残影池       | 清空，新赛季从冷启动 AI 开始，逐步被真实残影替代 |
| 玩家局内进度 | 无影响（每局本就是独立的）                       |

---

## 三、赛季保留内容

| 保留         | 说明     |
| ------------ | -------- |
| 累计成就     | 永久保留 |
| 历史最高记录 | 永久保留 |

---

## 四、赛季特色（每赛季轮换）

每个赛季可以引入 **1 个赛季规则**，增加新鲜感：

| 赛季规则示例 | 效果                                                  |
| ------------ | ----------------------------------------------------- |
| 三流派赛季   | 流派池从 2 个扩展为 3 个                              |
| 精英赛季     | 残影战斗对手 HP ×1.3，但奖励 ×1.5；野怪升级为精英野怪 |
| 遗物狂欢     | 遗物掉落概率翻倍，持有上限 +2                         |
| 速攻赛季     | 加时阈值从 100 回合降为 50 回合                       |

---

## 五、赛季排行榜

| 排行维度     | 说明                                               |
| ------------ | -------------------------------------------------- |
| 最高通关层数 | 主排行，同层比剩余 HP                              |
| 最速通关     | 通关第 10 层用时最短（实际操作时间，不含战斗动画） |
| 残影击杀榜   | 该赛季内作为残影被挑战的总次数 / 击杀其他玩家次数  |
