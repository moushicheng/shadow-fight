# 商店系统

## 一、出现时机

每层第 2 循环的**残影战斗结束后**进入商店（即每层仅 1 次商店机会）。野怪战斗后不进入商店，直接进入下一个事件节点。

---

## 二、商品栏位

| 栏位 | 数量   | 内容                            |
| ---- | ------ | ------------------------------- |
| 卡牌 | 5 张   | 流派池内 + 通用随机卡牌         |
| 药水 | 1-2 瓶 | 恢复类/增益类消耗品             |
| 遗物 | 0-1 个 | 50% 概率刷新，价格 100-250 金   |
| 服务 | 2 个   | 从 3 种服务中随机出现 2 种（§三） |

---

## 三、服务系统

### 3.1 服务种类

| 服务 | 说明 |
| ---- | ---- |
| **移除卡牌** | 从卡组中选 1 张删除。精简卡组是核心策略——初始 5 张普通攻击是负担，花钱删除是重要决策 |
| **升级卡牌** | 选 1 张卡，费用 -1 或效果数值 +30%（二选一） |
| **调整卡序** | 选 1 张卡牌，移动到卡组中的任意位置。诅咒卡不可调序（但可通过"移除卡牌"删除） |

### 3.2 服务规则

| 规则 | 说明 |
| ---- | ---- |
| 出现数量 | 每次进入商店，从 3 种服务中**随机出现 2 种** |
| 使用次数 | 每次进入商店**只能使用 1 次服务**（从出现的 2 种中选 1 种） |
| 基础价格 | 所有服务统一 **50 金**（首次） |
| 价格递增 | 每使用 1 次任意服务，**所有服务**价格永久 +25 金 |

### 3.3 价格阶梯示例

| 使用次数 | 当前服务价格 | 累计服务支出 |
| -------- | ------------ | ------------ |
| 第 1 次  | 50 金        | 50 金        |
| 第 2 次  | 75 金        | 125 金       |
| 第 3 次  | 100 金       | 225 金       |
| 第 4 次  | 125 金       | 350 金       |
| 第 5 次  | 150 金       | 500 金       |

### 3.4 卡牌获取与卡序

- 所有渠道（战斗奖励、商店购买、事件获得）获取的卡牌**自动追加到卡组末尾**
- 想要调整位置必须在商店使用"调整卡序"服务
- 诅咒卡（负面事件获得）同样追加到末尾，但**战斗开始时会被抽出并随机洗入卡组**（见 battle-base.md §四）

> **设计意图**：服务系统的三重稀缺性——随机出现（不一定有你想用的）、每次只能用一个（选了删除就不能调序）、越用越贵（前期便宜后期昂贵）——让每次商店都成为有意义的决策点。前期花 50 金把关键牌前移收益巨大，后期卡组精简后调序需求下降，自然形成不同阶段的金币分配策略。


## 四、宠物商店（待定，暂不设计
宠物是一种特殊的卡牌，在事件or商店中可获得，有了宠物之后能让宠物在作战中提供帮助，宠物拥有独立的属性和技能组，在作战中，敌人的行动会优先攻击宠物，超出宠物生命上限的伤害会反馈到主人身上。宠物死后可在下场战斗复活，并不是一次性的物品。
