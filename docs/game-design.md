# shadow-fight

## 一、游戏概述

**类型**：Roguelike 卡牌构筑 + 异步 PVP

**核心卖点**：随机属性、流派构筑、PvE + 异步 PvP——每局随机生成四维属性和可选流派池，通过爬塔中的事件、野怪战斗、残影对决、商店不断构筑卡组。每层先挑战野怪获取资源，再对抗其他玩家留下的"残影"数据。

**参考游戏**：杀戮尖塔（Slay the Spire）

**平台**：抖音小游戏（轻量级，单局 5-15 分钟）

---

## 二、核心循环

### 2.1 单层结构

每一层（Floor）包含 **2 个完整循环**，每个循环由 3 个节点组成。关键战斗节点前引入**玩家选择**，增加策略深度：

- **第 1 循环**（PvE）：事件 → **路线三选一**（精英战斗 / 普通战斗 / 未知事件） → 事件
- **第 2 循环**（PvP）：事件 → **残影三选一**（从 3 个残影中选择对手） → 商店

```
┌──────────────────────────── 第 N 层 ────────────────────────────┐
│                                                                  │
│  [事件] → [路线三选一] → [事件] → [事件] → [残影三选一] → [商店] │
│            ├ ⚔️精英战斗            ├ 👤残影A（可见概要）          │
│            ├ 🗡️普通战斗            ├ 👤残影B（可见概要）          │
│            └ 🎭未知事件            └ 👤残影C（可见概要）          │
│  ╰────── 第 1 循环（PvE） ──╯    ╰────── 第 2 循环（PvP） ──╯  │
│                                                                  │
│  * 若流派池含赌徒：战斗前额外触发「赌约」选择（可跳过）          │
│                                                                  │
│                       → 进入第 N+1 层                            │
└──────────────────────────────────────────────────────────────────┘
```

**设计意图**：路线选择让玩家根据当前 HP、卡组强度、金币储备做出风险/收益权衡；残影选择让玩家利用克制关系挑选有利对手。两个选择点将每层的决策密度从"仅事件选项"提升到"路线 + 对手 + 事件"三维决策，同时不显著增加操作时间（每次选择约 3-5 秒）。

### 2.2 层级推进

| 层数   | 难度定位 | 野怪战斗（第 1 循环）            | 残影战斗（第 2 循环）                           |
| ------ | -------- | -------------------------------- | ----------------------------------------------- |
| 1-3 层 | 新手区   | 弱小野怪，帮助玩家理解机制       | 低层残影，属性弱、卡组简单                      |
| 4-6 层 | 成长区   | 中等野怪，开始具备流派特色       | 中层残影，有完整流派构筑                        |
| 7-9 层 | 挑战区   | 精英野怪，拥有特殊技能和更高属性 | 高层残影，强度高，卡组精炼                      |
| 10 层  | Boss 层  | 无野怪战斗                       | Boss 残影：全服通关残影，1.5 倍 HP，无事件/商店 |

**单局流程**：玩家从第 1 层开始，每层通过路线选择和残影选择推进。死亡则本局结束、存档为残影供其他玩家挑战。通关第 10 层即为本局胜利。

### 2.3 路线选择（第 1 循环 · PvE）

每层第 1 循环的战斗节点不再固定，而是提供 **三选一路线**，让玩家根据当前状态做出风险/收益权衡：

| 路线 | 内容 | 奖励 | 适用场景 |
| --- | --- | --- | --- |
| ⚔️ 精英战斗 | 面对比当前层数更强的精英野怪 | 金币 ×1.5，必出稀有及以上卡牌，遗物掉率 25% | 卡组强、HP 充裕时冲高收益 |
| 🗡️ 普通战斗 | 标准野怪战斗（与原设计一致） | 标准奖励 | 稳健推进的默认选择 |
| 🎭 未知事件 | 跳过战斗，触发一个额外随机事件 | 无战斗奖励，但事件可能给予卡牌/遗物/金币/属性 | 残血保命、赌事件收益 |

**规则补充**：

- 1-2 层（新手引导期）锁定为普通战斗，不提供选择，帮助玩家熟悉战斗机制
- 3 层起解锁三选一
- 精英战斗的难度 = 当前层 +2 的野怪强度（如第 4 层精英 ≈ 第 6 层普通野怪）
- 未知事件从正面/中性/负面池中抽取，概率与当前层数的事件概率一致
- 偶尔（10% 概率）在三选一中出现第 4 个限时选项：💀 **赏金挑战**——Boss 级野怪，胜利奖励传说级卡牌/遗物，失败扣 30% 最大 HP

### 2.4 残影选择（第 2 循环 · PvP）

残影战斗不再随机匹配单个对手，而是从对手池中抽出 **3 个残影**展示概要信息，由玩家选择挑战目标：

**展示信息**：

| 信息 | 说明 |
| --- | --- |
| 玩家昵称/头像 | 社交展示 |
| 流派池 | 如「冰 + 刺客」，便于玩家判断克制关系 |
| 难度星级（★） | ★☆☆ 较弱 / ★★☆ 中等 / ★★★ 强力，根据残影的卡组完成度和属性总和计算 |
| 卡组张数 | 暗示构筑质量（少而精 vs 多而杂） |
| 特殊标记 | 「悬赏目标」额外 +50 金 /「复仇对象」优先展示 |

**不展示的信息**（保留悬念）：具体卡牌列表、遗物列表、精确属性数值。

**规则补充**：

- 难度星级越高，战斗奖励越好（高星残影金币 +30%、遗物掉率 +10%）
- 复仇名单中的残影优先出现在三选一中
- 好友残影优先出现在三选一中（社交联动）
- 若对手池残影不足 3 个，用预设 AI 补齐

## 三、角色系统

### 3.1 属性生成

每局开始时，系统使用**随机种子**生成四维基础属性：

| 属性 | 缩写 | 运行时映射               | 影响                           |
| ---- | ---- | ------------------------ | ------------------------------ |
| 力量 | STR  | attack = STR             | 物理伤害倍率                   |
| 体质 | CON  | HP = CON × 15 + 30       | 生命值                         |
| 速度 | SPD  | speed = SPD              | 行动次数、先手判定、刺客系倍率 |
| 法力 | MANA | mana = round(MANA × 1.5) | 每回合可用法力                 |

**生成规则**：

- 总点数固定为 **40 点**，在四维中随机分配
- SPD 下限为 3（避免 0 速度锁死）
- 玩家在开局看到属性后，可以选择**接受**或**重摇一次**（每局仅一次免费重摇）

### 3.2 运行时属性

| 属性             | 初始值            | 战斗间规则                                       |
| ---------------- | ----------------- | ------------------------------------------------ |
| HP               | CON × 15 + 30     | **跨战斗保留**，不自动回满                       |
| attack           | STR               | 可被事件/遗物/汲取修改                           |
| speed            | SPD               | 可被事件/遗物/霜蚀/汲取修改                      |
| mana             | round(MANA × 1.5) | 每场战斗重置为满，回合内每回合回复 1（不超上限） |
| armor            | 0                 | 每场战斗开始清零                                 |
| 霜蚀/灼烧/毒药   | 0                 | 每场战斗开始清零                                 |
| 诅咒卡（卡组中） | 无                | 每场战斗开始清零（诅咒卡从卡组中移除）           |
| 汲取效果         | 无                | 每场战斗开始清零（被偷的属性恢复）               |

### 3.3 HP 管理

- HP 在整局游戏中**持续计算**，战斗结束后不回满
- 恢复途径：商店药水、事件奖励、遗物被动
- 这是 Roguelike 的核心资源压力来源

## 四、流派池系统

### 4.1 核心机制

开局时系统从所有流派中随机抽取 **2 个流派**组成该角色的「流派池」。玩家在整局游戏中只能获取和使用**流派池内 + 通用**的卡牌。

### 4.2 流派总览

共 **10 个流派**，分为两类：

- **关键词流派**：拥有专属状态/关键词，围绕该状态构建卡牌体系
- **纯卡牌流派**：不引入新关键词，仅靠卡牌之间的数值配合和条件判断产生化学反应

| 流派   | 标签      | 类型       | 赢法定位               | 专属关键词 |
| ------ | --------- | ---------- | ---------------------- | ---------- |
| 冰系   | ICE       | 关键词流派 | 减速控场，行动碾压     | 霜蚀       |
| 火系   | FIRE      | 关键词流派 | 叠层引爆，一波秒杀     | 灼烧       |
| 毒系   | POISON    | 关键词流派 | 穿甲 DoT，慢性致死     | 毒药       |
| 咒术师 | HEX       | 关键词流派 | 污染对手卡组，干扰节奏 | 诅咒       |
| 血族   | BLOOD     | 关键词流派 | 偷属性滚雪球           | 汲取       |
| 刺客   | ASSASSIN  | 纯卡牌流派 | 堆速度多动，终结收割   | —          |
| 狂战士 | BERSERKER | 纯卡牌流派 | 越残越强，搏命输出     | —          |
| 守卫   | GUARDIAN  | 纯卡牌流派 | 堆甲防守，以守为攻     | —          |
| 武僧   | MONK      | 纯卡牌流派 | 法力精算，条件爆发     | —          |
| 赌徒   | GAMBLER   | 纯卡牌流派 | 高方差赌命             | —          |

**通用卡**不受流派池限制，任何角色都可以获取。

### 4.3 关键词流派详解

#### 冰系 — 霜蚀（控节奏）

**核心：不造成伤害，但剥夺对手的行动机会。**

- 霜蚀层数直接降低敌方速度：每 N 层霜蚀 = 敌方 speed -1
- 速度降低 → 行动次数减少（`floor(speed / minSpeed)`），对手每回合能打的牌变少
- 达到阈值触发「冻结」：敌方速度降到 1 以下时，跳过该回合所有行动
- 自然衰减：每回合 -2 层
- **不造成任何直接伤害**

策略体感：冰系不着急杀人，靠行动次数差碾压节奏。卡牌本身伤害不高，需要靠多动优势慢慢磨。

#### 火系 — 灼烧（打爆发）

**核心：灼烧本身不造成伤害，但让后续火系伤害越来越疼。**

- 灼烧层数 = 火系卡牌的额外伤害加成（如目标 6 层灼烧，10 伤害的火球实际造成 16）
- 部分卡牌可以「引爆」：消耗全部灼烧层数，造成 层数 × 倍率 的一次性爆发
- 灼烧不自然衰减，引爆后清零
- 非火系伤害不享受灼烧加成

策略体感：节奏是"蓄力 → 引爆"。前几张牌叠灼烧，后面引爆卡收割。卡序安排至关重要。

#### 毒系 — 毒药（磨穿肉盾）

**核心：持续掉血，且无视护甲。**

- 每回合结束，中毒方受到 = 毒药层数的伤害，**无视护甲**
- 每回合衰减 1 层
- 不影响速度、不影响伤害，纯粹的持续消耗

策略体感：最"不讲道理"的流派——堆再多护甲也没用。但毒不爆发，对手血厚恢复快就可能耗不死。

#### 咒术师 — 诅咒（干扰卡序）

**核心：往对方卡组里塞"垃圾牌"，破坏对手的出牌节奏。**

- 「诅咒卡」是负面卡牌，被塞进对手卡组后轮到时强制使用，浪费行动和法力
- 诅咒卡类型举例："费用 2，无效果"（纯浪费）、"费用 0，自伤 10"
- 部分卡牌可以替换/移除对方卡组中的正常卡牌
- 配合卡可以从对方的诅咒中获益（如"对方每打出 1 张诅咒卡，你回复 5 HP"）

策略体感：唯一直接干扰对手卡序的流派。对依赖卡序的流派（火系引爆、武僧节奏）是天然克制。

#### 血族 — 汲取（零和博弈）

**核心：偷取对手的属性来强化自己，差值是 ×2 效果。**

- 「汲取」：从对方身上扣除属性并加到自己身上（如汲取 2 ATK = 对方 -2，己方 +2）
- 可汲取的属性：攻击力、速度、护甲
- 部分卡牌附带吸血效果（造成伤害回复 HP）
- 汲取效果持续本场战斗

策略体感：战斗越久越强、对手越弱，形成滚雪球。但前期偷取量不够时是弱势方，需要扛过开局。

### 4.4 纯卡牌流派详解

#### 刺客 — 速度即正义

**设计思路**：卡牌大量涉及速度加成和速度缩放伤害。

- 加速类卡牌："SPD +5"、"将至多 8 点 STR 转为 SPD"
- 速度伤害卡："造成 1×SPD 伤害"
- 终结爆发卡："消耗全部 MP，每 1 MP 造成 0.5×SPD 伤害"
- 多次行动带来更多出牌 → 更快循环卡组

与属性的关系：天然偏好高 SPD 角色。

#### 狂战士 — 越残越强

**设计思路**：卡牌以「当前 HP」「已损失 HP」作为公式变量，不引入新状态。

- 低血增伤类："造成 (已损失 HP × 0.3) 伤害"
- HP 换伤害类："消耗 15 HP（不是 MP），造成 40 伤害"
- 条件爆发类："若 HP < 30%，本次伤害翻倍"

与 Roguelike 的互动：别的流派怕掉血，狂战士想掉血。事件中的"噩梦（扣 HP）"对狂战士可能是增益。HP 管理从"尽量保血"变为"精确控血线"。

#### 守卫 — 以甲为剑

**设计思路**：围绕现有「护甲」机制做文章，不引入新关键词。

- 堆甲类："获得 15 护甲"
- 甲转伤害类："造成 = 当前护甲值 的伤害"
- 甲引爆类："消耗全部护甲，造成 护甲值 × 2 伤害"
- 持续减伤类："若护甲 > 0，受到的伤害 -5"

节奏：每场战斗前几回合堆甲 → 中后期用甲值输出。卡序极为关键。

#### 武僧 — 法力即武器

**设计思路**：卡牌围绕「当前 MP」做条件判断和数值缩放。

- 高蓝条件卡："若当前 MP ≥ 8，造成 35 伤害；否则造成 10 伤害"
- 蓝量爆发卡："消耗所有 MP，每点 MP 造成 3 伤害"
- 回蓝卡："回复 MP 至上限，本回合无法攻击"
- 消费结算卡："费用 0，造成 = 本回合已消耗 MP × 2 的伤害"

策略体感：需要精确控制 MP 曲线——某些牌需要高蓝时放，某些牌需要刚好花完蓝后放。卡序编排像是在安排一段"呼吸节奏"。

#### 赌徒 — 命运骰子

**设计思路**：卡牌效果包含大量随机性，期望值偏高（高风险高回报）。

- 随机伤害卡："随机造成 0 / 10 / 20 / 50 伤害"
- 对赌卡："50% 概率造成 40 伤害，50% 概率自己受 15 伤害"
- 越界卡："随机获得一张不在你流派池中的卡牌（仅本场战斗）"
- 投骰卡："投骰子：结果 × 5 = 伤害"

社交属性：赌徒系是直播/分享时最有看点的流派，传播性强。适合作为"副流派"搭配其他流派使用。

#### 赌徒专属系统 — 赌约

赌徒是 10 个流派中唯一拥有**战斗外专属机制**的流派。当流派池包含 GAMBLER 时，每场战斗前会触发「赌约」选择界面（可一键跳过）。

**触发条件**：流派池包含 GAMBLER

**出现时机**：每场战斗前（野怪战斗和残影战斗均触发）

**机制**：系统从赌约池中随机展示 **2 个赌约**，玩家可选择接受 0-1 个，或直接跳过。

**基础赌约池**：

| 赌约 | 正面效果 | 负面代价 |
| --- | --- | --- |
| 自信一击 | 本场战斗 ATK +3 | 初始 HP -20 |
| 贪婪协议 | 金币奖励 ×2 | 卡牌奖励从 3 选 1 变为 2 选 1 |
| 命运硬币 | 50% 概率额外获得 1 个遗物选择 | 50% 概率失去 50 金 |
| 玻璃之躯 | 本场战斗伤害 +50% | 本场战斗受到伤害 +50% |
| 速战速决 | 本场战斗 SPD +5 | 超过 30 回合未结束则直接判负 |
| 孤注一掷 | 胜利后恢复 30% 最大 HP | 失败后额外扣除 20% 最大 HP |

**遗物联动**（赌徒专属遗物，强化赌约系统）：

| 遗物 | 品质 | 效果 |
| --- | --- | --- |
| 赌神骰子 | 稀有 | 赌约展示数量从 2 个增加到 3 个 |
| 幸运硬币 | 稀有 | 赌约中所有概率性负面效果的触发概率减半 |
| 豪赌之心 | 传说 | 每场战斗可同时接受 2 个赌约 |

> **设计意图**：赌约系统让赌徒流派的"赌"从卡牌层面延伸到整个闯关体验。其他流派的玩家走标准流程，而赌徒玩家每场战斗前都面临一个"要不要加注"的额外决策——这让赌徒成为唯一能改变游戏节奏本身的流派，而非仅仅是"卡牌效果随机"。同时，由于赌约可以一键跳过，不会对不喜欢赌博的玩家造成操作负担。

### 4.5 流派池规则

| 环节            | 影响                        |
| --------------- | --------------------------- |
| 战斗奖励 3 选 1 | 只从流派池内 + 通用卡中随机 |
| 商店卡牌栏      | 只刷流派池内 + 通用卡       |
| 事件获得卡牌    | 只从流派池内 + 通用卡中随机 |

### 4.6 流派组合示例

10 个流派取 2 个，共 C(10,2) = **45 种组合**，以下列举部分有代表性的搭配思路：

| 组合        | 玩法倾向                                                              |
| ----------- | --------------------------------------------------------------------- |
| 冰 + 毒     | 冰减速拖时间，毒持续结算，慢性窒息                                    |
| 火 + 刺客   | 速度多动快速叠灼烧，引爆一波带走                                      |
| 毒 + 守卫   | 堆甲扛伤，毒穿甲慢磨对手，自己不怕毒（有甲…等等毒无视甲，这对镜很痛） |
| 狂战 + 血族 | 吸血回复与低血增伤的微妙平衡                                          |
| 咒术 + 冰   | 双重控制——速度被冰减，卡序被诅咒打乱                                  |
| 武僧 + 火   | 气定神闲（0 费）配合武僧"已消耗 MP"结算卡                             |
| 赌徒 + 任意 | 高方差副流派，关键时刻赌一把翻盘                                      |
| 守卫 + 血族 | 堆甲扛过前期，汲取属性滚雪球                                          |
| 刺客 + 武僧 | 双终结体系，速度保证出牌量，法力精算保证爆发                          |
| 咒术 + 毒   | 诅咒拖慢对手节奏，毒在背景中持续消耗                                  |

### 4.7 流派间的克制关系

```
        冰 ──克制──→ 刺客（减速废掉速度体系）
        毒 ──克制──→ 守卫（无视护甲）
      咒术 ──克制──→ 火/武僧（打乱卡序 = 打乱蓄力/节奏）
      刺客 ──克制──→ 毒（速度优势快速击杀，不给毒结算时间）
      守卫 ──克制──→ 狂战（高甲扛伤，狂战打不动）
      狂战 ──克制──→ 冰（冰不造成伤害，狂战不怕被控只怕被秒）
      血族 ──克制──→ 守卫/武僧（偷属性让精密体系崩溃）
      火系 ──克制──→ 血族（引爆秒杀不给滚雪球时间）
```

> 克制关系不是绝对的，卡组构筑和遗物配合可以弥补劣势。但开局看到流派池时，了解克制关系有助于判断该侧重哪个流派。

> **策划意图**：C(10,2)=45 种组合足够支撑多个赛季的内容。玩家看到流派池 + 四维属性后需要做出判断——侧重哪个流派、如何搭配，这是每局开始的第一个策略决策点。

## 六、遗物系统

### 6.1 概述

遗物是**局内持续生效的被动道具**，一旦获得，效果持续到本局结束。

### 6.2 获取途径

| 途径      | 说明                                                   |
| --------- | ------------------------------------------------------ |
| 战斗掉落  | 每场战斗胜利有 **20%** 概率额外掉落 1 个遗物（3 选 1） |
| 事件奖励  | 部分事件给予遗物                                       |
| 商店购买  | 商店有概率刷新 1 个遗物（价格 100-250 金）             |
| Boss 必掉 | 击败 Boss 必定获得 1 个遗物                            |

### 6.3 遗物品质

| 品质 | 获得概率 | 效果强度                   |
| ---- | -------- | -------------------------- |
| 普通 | 60%      | 小幅被动增益               |
| 稀有 | 30%      | 中等被动增益或条件触发效果 |
| 传说 | 10%      | 改变构筑方向的强力效果     |

### 6.4 遗物持有上限

玩家最多同时持有 **5 个遗物**。超出时必须选择丢弃一个。

### 6.5 遗物示例（各品质 1 个参考，完整列表后续设计）

| 遗物名     | 品质 | 效果                                               |
| ---------- | ---- | -------------------------------------------------- |
| 铁皮水壶   | 普通 | 每场战斗开始时获得 5 护甲                          |
| 红宝石戒指 | 稀有 | 每次进入商店时恢复 10 HP                           |
| 元素共鸣   | 传说 | 当卡组中同流派卡牌 ≥ 5 张时，该流派所有卡牌费用 -1 |

> **策划意图**：遗物系统是卡牌之外的第二构筑维度。「元素共鸣」类遗物鼓励深入单一流派，与流派池机制形成配合。遗物的随机性也是 Roguelike 每局体验差异化的重要来源。

## 七、战斗系统

### 7.1 完全自动战斗

战斗中玩家**不做任何操作**，系统按卡组顺序自动出牌。玩家的所有策略决策发生在战斗之外（卡组构筑、卡序调整、商店购买、事件选择）。

UI 提供「加速」和「跳过」按钮，方便快速看结果。

### 7.2 战斗流程

```
每回合：roundStart → prepare（双方按卡序出牌） → rounding（攻击执行） → roundEnd（状态结算）
```

**行动次数**：`floor(己方速度 / min(双方速度))`，速度优势方每回合多出牌。

**先手判定**：行动次数多者先手；相同则速度高者先手。

### 7.3 战斗结束条件

- 任一方 HP ≤ 0 → 该方死亡
- 超过 200 回合 → 超时平局（双方均不获得奖励）
- 100 回合后进入加时：每回合双方受 `1 + (回合数 - 100) × 2` 伤害

### 7.4 状态效果

#### 关键词状态

| 状态 | 来源   | 造成伤害？ | 核心效果                                                       | 衰减方式           |
| ---- | ------ | ---------- | -------------------------------------------------------------- | ------------------ |
| 霜蚀 | 冰系   | 否         | 降低敌方速度（每 N 层 = -1 SPD），速度 ≤0 时冻结跳过回合       | 每回合 -2 层       |
| 灼烧 | 火系   | 间接       | 火系卡牌获得额外伤害 = 目标灼烧层数；可被「引爆」一次性结算    | 不衰减，引爆后清零 |
| 毒药 | 毒系   | 是         | 每回合结束按层数扣血，**无视护甲**                             | 每回合 -1 层       |
| 诅咒 | 咒术师 | —          | 负面卡牌被塞入对手卡组，轮到时强制使用，浪费行动/法力/造成自伤 | 被打出后移除       |
| 汲取 | 血族   | —          | 从对方扣除属性（ATK/SPD/护甲）并加到自己身上，持续本场战斗     | 不衰减             |

#### 通用状态

| 状态 | 来源   | 机制                                    |
| ---- | ------ | --------------------------------------- |
| 护甲 | 多流派 | 受击时先扣护甲再扣 HP，每场战斗开始清零 |

### 7.5 战斗奖励

#### 野怪战斗奖励（第 1 循环）

| 奖励 | 内容                                       |
| ---- | ------------------------------------------ |
| 金币 | 基础 10-20 金 + 层数加成（每层 +3）        |
| 卡牌 | 从流派池 + 通用中随机 3 张，选 1（可跳过） |
| 遗物 | 10% 概率触发，随机 3 个选 1                |

#### 残影战斗奖励（第 2 循环）

| 奖励 | 内容                                       |
| ---- | ------------------------------------------ |
| 金币 | 基础 20-30 金 + 层数加成（每层 +5）        |
| 卡牌 | 从流派池 + 通用中随机 3 张，选 1（可跳过） |
| 遗物 | 25% 概率触发，随机 3 个选 1                |

> **设计意图**：残影战斗难度更高，对应奖励也更丰厚。野怪战斗提供基础资源积累，残影战斗是主要的卡牌和遗物获取来源。

## 八、野怪系统

### 8.1 概述

野怪是每层第 1 循环的 PvE 对手，由系统预设生成。野怪战斗为玩家提供资源积累和卡组测试的机会，难度低于同层残影。

### 8.2 野怪生成规则

| 属性     | 规则                                                           |
| -------- | -------------------------------------------------------------- |
| 基础属性 | 根据层数按预设公式生成，总点数 = 25 + 层数 × 3                 |
| 卡组     | 从预设野怪卡池中按层数和类型组装，卡组大小 = 3 + 层数          |
| 遗物     | 无遗物（1-3 层）/ 1 个普通遗物（4-6 层）/ 1-2 个遗物（7-9 层） |
| HP       | 按层数难度系数计算，整体低于同层残影                           |

### 8.3 野怪类型

每层的野怪从对应难度区间的野怪池中随机抽取，具备不同的战斗风格：

| 类型     | 特点                                | 出现层数 |
| -------- | ----------------------------------- | -------- |
| 普通野怪 | 基础属性平均，使用通用卡牌          | 1-10 层  |
| 流派野怪 | 具备某一流派的简易卡组              | 3-10 层  |
| 精英野怪 | 属性更高，拥有 1 个遗物，卡组更完善 | 6-10 层  |

### 8.4 野怪与残影的对比

| 维度     | 野怪（第 1 循环）   | 残影（第 2 循环）                  |
| -------- | ------------------- | ---------------------------------- |
| 来源     | 系统预设生成        | 其他玩家的死亡存档                 |
| 难度     | 较低，侧重资源积累  | 较高，考验构筑能力                 |
| 社交属性 | 无                  | 显示对手昵称/头像，支持复仇        |
| 奖励     | 基础金币 + 卡牌选择 | 更高金币 + 卡牌选择 + 更高遗物概率 |
| 战后节点 | 进入第 2 循环事件   | 进入商店                           |

> **策划意图**：野怪作为 PvE 缓冲层，降低了每层的整体挫败感。玩家即使在残影战斗中失败，至少在野怪战斗中获得了一些资源和成长。同时野怪战斗也是新手熟悉机制的安全空间。

## 九、异步 PVP — 残影系统

### 9.1 核心机制

- 玩家死亡时，系统记录其**死亡层数 + 当前卡组 + 基础属性 + 遗物列表**
- 该数据作为「残影」存入对应层的**对手池**
- 残影**仅出现在每层的第 2 循环（残影战斗节点）**，第 1 循环为野怪战斗
- 其他玩家打到该层的第 2 循环时，从对手池中随机匹配残影作为对手
- 残影的战斗逻辑与真人完全一致（按卡序自动出牌）

### 9.2 残影数据结构

```typescript
type Ghost = {
  oderId: string;
  playerName: string;
  seed: number; // 随机种子（用于重建属性）
  baseProperty: {
    // 死亡时的基础属性（含事件修改）
    STR: number;
    CON: number;
    SPD: number;
    MANA: number;
  };
  deck: string[]; // 有序卡组
  relics: string[]; // 遗物列表
  floor: number; // 死在第几层
  factionPool: string[]; // 流派池
  timestamp: number;
};
```

### 9.3 对手池与三选一匹配规则

残影战斗采用**三选一**机制（详见 2.4 节），从对手池中抽出 3 个残影供玩家选择。

| 规则     | 说明                                                              |
| -------- | ----------------------------------------------------------------- |
| 抽取数量 | 从同层残影池中抽取 **3 个**残影，展示概要后由玩家选择 1 个挑战    |
| 难度分布 | 3 个残影尽量覆盖不同难度星级（1 弱 + 1 中 + 1 强），避免三个同档 |
| HP 规则  | 残影 HP = maxHp × 层数难度系数（不用死亡时 HP，避免太弱）        |
| 优先展示 | 复仇名单、好友残影在 3 个候选位中优先占位                        |
| 冷启动   | 残影不足 3 个时使用预设 AI 补齐                                   |
| 数据过期 | 残影保留**当前赛季**内有效                                        |
| 去重     | 同一局不会遇到同一残影两次                                        |

## 十一、商店系统

### 11.1 出现时机

每层第 2 循环的**残影战斗结束后**进入商店（即每层仅 1 次商店机会）。野怪战斗后不进入商店，直接进入下一个事件节点。

### 11.2 商品栏位

| 栏位 | 数量   | 内容                                                   |
| ---- | ------ | ------------------------------------------------------ |
| 卡牌 | 3 张   | 流派池内 + 通用随机卡牌                                |
| 药水 | 1-2 瓶 | 恢复类/增益类消耗品                                    |
| 遗物 | 0-1 个 | 50% 概率刷新，价格 100-250 金                          |
| 服务 | 常驻   | 移除卡牌（50 金）、卡牌升级（80 金）、调整卡序（免费） |

### 11.3 关键服务说明

- **移除卡牌**：从卡组中选 1 张删除。精简卡组是核心策略——初始 5 张普通攻击是负担，花钱删除是重要决策
- **卡牌升级**：选 1 张卡，费用 -1 或效果数值 +30%（二选一）
- **调整卡序**：免费，重新排列卡组中卡牌的顺序。只在商店可用，强调规划意识

```
│  └───────────────┘  │
│  ┌─┐ ┌─┐ ┌─┐ ┌─┐  │
│  │1│ │2│ │3│ │4│  │  ← 卡组队列预览（接下来 4 张）

│    [加速]  [跳过]   │  ← 操作按钮
└─────────────────────┘
```

### 13.3 单局时长控制

- 目标：**5-15 分钟**
- 战斗完全自动，玩家决策点在战斗外：**路线选择、残影选择**、事件选择、卡牌奖励 3 选 1、商店购买、卡组排序、赌约选择（赌徒流派）
- 路线选择和残影选择每次约 3-5 秒，不显著增加单局时长
- 支持中途暂停/恢复（存当前 RunState）
