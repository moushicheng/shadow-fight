# 核心循环

## 一、单层结构

每一层（Floor）包含 **2 个完整循环**，每个循环由 3 个节点组成。关键战斗节点前引入**玩家选择**，增加策略深度：

- **第 1 循环**（PvE）：事件 → **路线三选一**（精英战斗 / 普通战斗 A / 普通战斗 B） → 事件
- **第 2 循环**（PvP）：事件 → **残影三选一**（从 3 个残影中选择对手） → 商店

```
┌──────────────────────────── 第 N 层 ────────────────────────────┐
│                                                                  │
│  [事件] → [路线三选一] → [事件] → [事件] → [残影三选一] → [商店] │
│            ├ ⚔️精英战斗（显示怪物名） ├ 👤残影A（可见概要）      │
│            ├ 🗡️普通战斗A（显示怪物名） ├ 👤残影B（可见概要）     │
│            └ 🗡️普通战斗B（显示怪物名） └ 👤残影C（可见概要）     │
│  ╰────── 第 1 循环（PvE） ──╯    ╰────── 第 2 循环（PvP） ──╯  │
│                                                                  │
│  * 若流派池含赌徒：战斗前额外触发「赌约」选择（可跳过）          │
│  * 10% 概率额外出现第 4 个选项：💀赏金挑战                       │
│                                                                  │
│                       → 进入第 N+1 层                            │
└──────────────────────────────────────────────────────────────────┘
```

**设计意图**：路线选择让玩家根据当前 HP、卡组强度、金币储备做出风险/收益权衡；每个选项展示怪物名称，让玩家基于已知信息（怪物流派、类型）做出决策。残影选择让玩家利用克制关系挑选有利对手。两个选择点将每层的决策密度从"仅事件选项"提升到"路线 + 对手 + 事件"三维决策，同时不显著增加操作时间（每次选择约 3-5 秒）。

---

## 二、生命机制（心）

玩家每局拥有 **5 颗心**，代表可承受的失败次数：

- 战斗中 HP 归零 **不会立刻结束本局**，而是消耗 1 颗心，结束当前战斗，进入下一个流程节点
- 心耗尽（0 颗）时本局真正结束，存档为残影供其他玩家挑战
- 心不可恢复（整局固定 5 颗，不受事件/遗物影响）
- 战斗失败后，玩家 HP 恢复至最大 HP 的一定比例（可配置，暂定 100%）

**设计意图**：降低单次失败的挫败感，鼓励玩家挑战高难度路线（精英、赏金），让"冒险"成为合理的策略选项而非单纯的赌博。

---

## 三、层级推进

| 层数   | 难度定位 | 野怪战斗（第 1 循环）            | 残影战斗（第 2 循环）                           |
| ------ | -------- | -------------------------------- | ----------------------------------------------- |
| 1-3 层 | 新手区   | 弱小野怪，帮助玩家理解机制       | 低层残影，属性弱、卡组简单                      |
| 4-6 层 | 成长区   | 中等野怪，开始具备流派特色       | 中层残影，有完整流派构筑                        |
| 7-9 层 | 挑战区   | 精英野怪，拥有特殊技能和更高属性 | 高层残影，强度高，卡组精炼                      |
| 10 层  | Boss 层  | 无野怪战斗                       | Boss 残影：全服通关残影，1.5 倍 HP，无事件/商店 |

**单局流程**：玩家从第 1 层开始，每层通过路线选择和残影选择推进。心耗尽则本局结束、存档为残影供其他玩家挑战。通关第 10 层即为本局胜利。

## 四、路线选择（第 1 循环 · PvE）

每层第 1 循环的战斗节点不再固定，而是提供 **三选一路线**，每个选项展示怪物名称，让玩家根据当前状态和对手信息做出风险/收益权衡：

| 路线 | 内容 | 奖励 | 适用场景 |
| --- | --- | --- | --- |
| ⚔️ 精英战斗 | 面对比当前层数更强的精英野怪（显示怪物名称） | 金币 ×1.5，必出稀有及以上卡牌，遗物掉率 25% | 卡组强、心充裕时冲高收益 |
| 🗡️ 普通战斗 A | 标准野怪战斗（显示怪物名称） | 标准奖励 | 稳健推进的默认选择 |
| 🗡️ 普通战斗 B | 另一个标准野怪战斗（显示怪物名称，与 A 不同） | 标准奖励 | 根据怪物流派/类型选择更有利的对手 |

**展示信息**：每个路线选项展示怪物名称（如「冰霜哨兵」「毒沼蛙」），让两个普通战斗选项有区分度，玩家可根据怪物流派做出决策。

**规则补充**：

- 1-2 层（新手引导期）锁定为普通战斗，不提供选择，帮助玩家熟悉战斗机制
- 3 层起解锁三选一
- 精英战斗的难度 = 当前层 +2 的野怪强度（如第 4 层精英 ≈ 第 6 层普通野怪）
- 两个普通战斗选项从当前层的可用野怪池中分别抽取，确保怪物不同
- 偶尔（10% 概率）在三选一中出现第 4 个选项：💀 **赏金挑战**
  - 赏金怪物强度 = 精英参数 × 1.5（属性总点 × 1.5，卡组按精英等效层数）
  - 胜利奖励 = 标准奖励 + 额外随机获得 1 个遗物
  - 失败消耗 1 颗心（与普通战斗失败一致）

---

## 五、残影选择（第 2 循环 · PvP）

残影战斗不再随机匹配单个对手，而是从对手池中抽出 **3 个残影**展示概要信息，由玩家选择挑战目标：

**展示信息**：

| 信息 | 说明 |
| --- | --- |
| 玩家昵称/头像 | 社交展示 |
| 流派池 | 如「冰 + 刺客」，便于玩家判断克制关系 |
| 难度星级（★） | ★☆☆ 较弱 / ★★☆ 中等 / ★★★ 强力，根据残影的卡组完成度和属性总和计算 |
| 卡组张数 | 暗示构筑质量（少而精 vs 多而杂） |
| 特殊标记 | 「悬赏目标」额外 +50 金 /「复仇对象」优先展示 |

**不展示的信息**（保留悬念）：具体卡牌列表、遗物列表、精确属性数值。

**规则补充**：

- 难度星级越高，战斗奖励越好（高星残影金币 +30%、遗物掉率 +10%）
- 复仇名单中的残影优先出现在三选一中
- 好友残影优先出现在三选一中（社交联动）
- 若对手池残影不足 3 个，用预设 AI 补齐
