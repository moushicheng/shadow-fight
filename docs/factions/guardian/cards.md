# 守卫 (GUARDIAN) — 卡牌设计

## 派系定位

**纯卡牌流派** | 赢法：堆甲防守，以守为攻

## 设计思路

围绕现有「护甲」机制做文章，不引入新关键词。护甲既是防御资源也是输出资源。

核心体验是"前几回合堆甲站住 → 甲值够高后转攻 → 用护甲值碾压对手"。

### 卡牌设计方向

1. **堆甲类** — 高效获取护甲
2. **甲转伤害类** — 以当前护甲值为变量的伤害牌
3. **甲引爆类** — 消耗全部护甲，倍率转化为伤害
4. **持续减伤类** — 有甲时获得额外减伤效果
5. **甲值维持类** — 防止护甲被消耗过快或回复护甲

### 节奏特点

每场战斗前几回合堆甲 → 中后期用甲值输出。卡序极为关键——堆甲牌必须排在甲转伤害牌前面。

### 属性倾向

不强依赖特定属性，但高 CON（高血量）配合高护甲会非常肉，高 SPD 有利于更快堆起护甲。

### 克制关系

- **克制** → 狂战士（高甲扛伤，狂战打不动）
- **被克** ← 毒系（无视护甲）、血族（偷护甲让体系崩溃）
