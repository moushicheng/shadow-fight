# 毒系 (POISON) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：穿甲 DoT，慢性致死

## 专属关键词：毒药

- 每回合结束，中毒方受到 = 毒药层数的伤害，**无视护甲**
- 每回合衰减 1 层
- 不影响速度、不影响伤害，纯粹的持续消耗

## 设计思路

最"不讲道理"的流派——堆再多护甲也没用。但毒不爆发，对手血厚恢复快就可能耗不死。

核心体验是"让毒持续跳 → 维持高层数不断档 → 对手慢慢被毒死"。

### 卡牌设计方向

1. **施毒类** — 高效叠加毒药层数
2. **维持类** — 减缓毒药衰减或回复毒药层数
3. **毒量爆发类** — 根据当前毒药层数获得额外效果
4. **防御拖延类** — 提供生存手段，拉长战斗让毒持续结算

### 属性倾向

不依赖特定属性，但高 CON（高血量）有助于拖长战斗让毒充分结算。

### 克制关系

- **克制** → 守卫（无视护甲）
- **被克** ← 刺客（速度优势快速击杀，不给毒结算时间）
