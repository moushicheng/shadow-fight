# 咒术师 (HEX) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：污染对手卡组，干扰节奏

## 专属关键词：诅咒

- 「诅咒卡」是负面卡牌，被塞进对手卡组后轮到时强制使用，浪费行动和法力
- 诅咒卡类型举例："费用 2，无效果"（纯浪费）、"费用 0，自伤 10"
- 部分卡牌可以替换/移除对方卡组中的正常卡牌
- 配合卡可以从对方的诅咒中获益（如"对方每打出 1 张诅咒卡，你回复 5 HP"）

## 设计思路

唯一直接干扰对手卡序的流派。对依赖卡序的流派（火系引爆、武僧节奏）是天然克制。

核心体验是"往对手卡组里不停塞垃圾 → 对手出牌节奏全乱 → 趁乱打输出"。

### 卡牌设计方向

1. **塞诅咒类** — 向对手卡组插入各种负面诅咒卡
2. **诅咒获益类** — 当对手打出诅咒卡时获得回报
3. **卡组干扰类** — 替换/移除对手卡组中的正常卡
4. **自身输出类** — 在诅咒体系之外提供基础伤害手段

### 诅咒卡类型（需单独设计 5-8 种）

诅咒卡是塞入对手卡组的负面牌，需要多样化设计以增加干扰维度。

### 属性倾向

不强依赖特定属性，但高 MANA 有助于持续施放诅咒牌。

### 克制关系

- **克制** → 火系/武僧（打乱卡序 = 打乱蓄力/节奏）
- **被克** ← 无明确被克（但卡组较大的对手受影响较小）
