# 狂战士 (BERSERKER) — 卡牌设计

## 派系定位

**纯卡牌流派** | 赢法：越残越强，搏命输出

## 设计思路

卡牌以「当前 HP」「已损失 HP」作为公式变量，不引入新状态。别的流派怕掉血，狂战士想掉血。

核心体验是"血量越低伤害越高 → 精确控血线 → 在死亡边缘爆发最大输出"。

### 卡牌设计方向

1. **低血增伤类** — 以已损失 HP 为变量计算伤害
2. **HP 换伤害类** — 消耗自身 HP（不是 MP）造成高额伤害
3. **条件爆发类** — HP 低于阈值时效果翻倍或触发额外效果
4. **自伤触发类** — 主动降低己方 HP 来激活各种条件
5. **绝境反击类** — 濒死时的保命/翻盘手段

### 与 Roguelike 的互动

事件中的"噩梦（扣 HP）"对狂战士可能是增益。HP 管理从"尽量保血"变为"精确控血线"——这是独特的 Roguelike 体验。

### 属性倾向

偏好高 CON（更大的 HP 池 = 更大的"已损失 HP"天花板），高 STR 也有基础伤害加成。

### 克制关系

- **克制** → 冰系（冰不造成伤害，狂战不怕被控只怕被秒）
- **被克** ← 守卫（高甲扛伤，狂战打不动）
