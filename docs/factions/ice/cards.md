# 冰系 (ICE) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：减速控场，行动碾压

## 专属关键词：霜蚀

- 霜蚀层数直接降低敌方速度：每 3 层霜蚀 = 敌方 effective_speed -1
- 速度降低 → ATB 行动槽填充变慢，对手出牌频率降低
- 达到阈值触发「冻结」：敌方 effective_speed ≤ 0 时，行动槽停止增长，无法行动
- 自然衰减：每周期 -1 层
- **不造成任何直接伤害**（霜蚀本身不伤血，伤害全靠卡牌）

## 设计思路

冰系不着急杀人，靠行动次数差碾压节奏。卡牌本身伤害不高，需要靠多动优势慢慢磨。

核心体验是"冻住对手 → 我比你多出牌 → 用行动数量优势慢慢碾死你"。

### 卡牌设计方向

1. **施加霜蚀类** — 主要手段，快速叠层降速
2. **冻结触发类** — 当敌方处于冻结时获得额外收益
3. **持续控制类** — 能力牌持续施加霜蚀，减缓衰减
4. **借速输出类** — 利用霜蚀层数 / 速度优势进行伤害转化

### 属性倾向

天然偏好高 SPD 角色：己方速度越高、对手速度越低，行动次数差越大。

### 克制关系

- **克制** → 刺客（减速废掉速度体系）
- **被克** ← 狂战士（冰系伤害偏低，狂战不怕被控只怕被秒）

### 数值设计锚点

- SPD 10 的目标需要 30 层霜蚀才能冻结（30/3=10, speed 10-10=0）
- 霜蚀每周期 -1 层，所以要维持冻结需要持续施加
- 白卡霜蚀量（8层 / 3层）是基础节奏，蓝卡（15层）可明显减速，紫卡（24层）接近冻结线
- 冰系卡牌伤害普遍低于同费用通用卡（用控制换伤害）

## 卡牌详细设计

### 丢雪球

- **ID**：ice_snowball
- **流派**：ICE
- **品质**：白（NORMAL）
- **类型**：技能（SKILL）
- **费用**：2
- **标签**：FREEZE, CHARGE
- **出现层数**：1-10
- **掉落权重**：1.0
- **效果描述**：造成 2 点伤害，施加 8 层霜蚀（减速 2）
- **效果详情**：
  - `{ target: "ENEMY", damage: { base: 2 }, status: { type: "FROST", stacks: 8 } }`
- **升级路线 A（费用 -1）**：费用降至 1，其余不变
- **升级路线 B（效果 +30%）**：3 伤害，施加 10 层霜蚀（减速 3）
- **设计意图**：冰系面包黄油牌，主要目标是快速叠霜蚀层数。伤害几乎可以忽略，但 8 层霜蚀 = 减速 2，是冰系前期的核心节奏牌。2 费保证不会被滥用

## 构筑指南

### 卡序建议

```
[冰封领域] → [丢雪球/寒风凛冽] → [极寒风暴] → [绝对零度] → [冰晶碎击/寒霜之刃 ×N]
  能力牌先出      中期叠霜蚀          大量霜蚀→冻结     冻结收割          循环输出
```

### 核心卡组骨架

- **控制核心**：冰封领域 + 寒风凛冽 / 极寒风暴
- **输出核心**：冰晶碎击 + 绝对零度
- **填充牌**：寒霜之刃（低费稳定输出）、寒冰护盾（防御）
- **梦幻单卡**：永冬（获得即胜势）

### 搭配建议

- **冰 + 守卫**：堆甲 + 减速，极端防御流。靠行动优势和冰晶碎击慢慢磨
- **冰 + 毒系**：减速 + DoT，双重持续伤害。敌方被减速后无法快速击杀你，毒药持续滴血
- **冰 + 血族**：减速 + 汲取 SPD，双重减速效果叠加。一旦成型对方几乎无法行动
