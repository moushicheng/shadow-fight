# 冰系 (ICE) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：减速控场，行动碾压

## 专属关键词：霜蚀

- 霜蚀层数直接降低敌方速度：每 N 层霜蚀 = 敌方 speed -1
- 速度降低 → 行动次数减少（`floor(speed / minSpeed)`），对手每回合能打的牌变少
- 达到阈值触发「冻结」：敌方速度降到 1 以下时，跳过该回合所有行动
- 自然衰减：每回合 -2 层
- **不造成任何直接伤害**

## 设计思路

冰系不着急杀人，靠行动次数差碾压节奏。卡牌本身伤害不高，需要靠多动优势慢慢磨。

核心体验是"冻住对手 → 我比你多出牌 → 用行动数量优势慢慢碾死你"。

### 卡牌设计方向

1. **施加霜蚀类** — 主要手段，快速叠层降速
2. **冻结触发类** — 当敌方处于冻结时获得额外收益
3. **持续控制类** — 减缓霜蚀衰减或持续施加
4. **借速输出类** — 利用速度差进行伤害转化

### 属性倾向

天然偏好高 SPD 角色：己方速度越高、对手速度越低，行动次数差越大。

### 克制关系

- **克制** → 刺客（减速废掉速度体系）
- **被克** ← 狂战士（冰不造成伤害，狂战不怕被控只怕被秒）
