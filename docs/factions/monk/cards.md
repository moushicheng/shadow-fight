# 武僧 (MONK) — 卡牌设计

## 派系定位

**纯卡牌流派** | 赢法：法力精算，条件爆发

## 设计思路

卡牌围绕「当前 MP」做条件判断和数值缩放。需要精确控制 MP 曲线——某些牌需要高蓝时放，某些牌需要刚好花完蓝后放。

核心体验是"精确控制 MP 曲线 → 高蓝牌和低蓝牌交替 → 像呼吸一样有节奏地爆发"。

### 卡牌设计方向

1. **高蓝条件类** — MP 高于阈值时效果大幅增强
2. **蓝量爆发类** — 消耗所有 MP 转化为伤害
3. **回蓝类** — 回复 MP 但附带限制（如无法攻击）
4. **消费结算类** — 根据本回合已消耗 MP 计算效果
5. **蓝量维持类** — 在出牌过程中保持 MP 在理想区间

### 节奏特点

卡序编排像是在安排一段"呼吸节奏"：回蓝 → 高蓝条件牌 → 花蓝 → 消费结算牌 → 再回蓝。

### 属性倾向

天然偏好高 MANA 角色（更大的 MP 池 = 更高的蓝量爆发天花板）。

### 克制关系

- **克制** → 无明确强克（但对低速对手有节奏优势）
- **被克** ← 咒术师（打乱卡序破坏 MP 节奏）、血族（偷属性让精密体系崩溃）
