# 刺客 (ASSASSIN) — 卡牌设计

## 派系定位

**纯卡牌流派** | 赢法：堆速度多动，终结收割

## 设计思路

卡牌大量涉及速度加成和速度缩放伤害。速度即正义——速度越高，行动次数越多，每张牌的伤害也越高。

核心体验是"疯狂加速 → 一回合出一堆牌 → 每张牌都按速度算伤害 → 暴风骤雨"。

### 卡牌设计方向

1. **加速类** — 提升己方 SPD，直接增加行动次数
2. **属性转化类** — 将 STR 等属性转为 SPD，极端化速度
3. **速度伤害类** — 以当前 SPD 为公式变量的伤害牌
4. **终结爆发类** — 消耗资源换取基于 SPD 的巨额伤害
5. **多动奖励类** — 同回合内出牌越多越强的效果

### 属性倾向

天然偏好高 SPD 角色。多次行动带来更多出牌 → 更快循环卡组。

### 克制关系

- **克制** → 毒系（速度优势快速击杀，不给毒结算时间）
- **被克** ← 冰系（减速废掉速度体系）
