# 通用 (COMMON) — 卡牌设计

## 定位

通用卡**不受流派池限制**，任何角色都可以获取。提供基础战斗能力和通用工具。

## 设计思路

通用卡是所有构筑的底座，提供最基础的攻击、防御和资源管理。不应比任何流派的专属卡更强，但要足够灵活以在任何组合中发挥作用。

核心体验是"可靠的基础工具 → 任何流派都能用 → 不强但不拖后腿"。

### 卡牌设计方向

1. **基础攻击类** — 朴实的伤害牌（如普通攻击）
2. **基础防御类** — 简单的护甲/减伤
3. **资源管理类** — MP 回复、抽牌加速等工具牌
4. **通用增益类** — 小幅属性提升或临时 buff

### 特殊说明

- 初始卡组的 5 张「普通攻击」属于通用卡
- 通用卡是玩家在商店中总能看到的选项
- 设计上偏保守，避免通用卡过强导致流派卡失去意义

## 关键词

### 变化（TRANSFORM）

将卡组中的卡牌随机替换为当前流派池内的其他卡牌。变化后的卡牌保留在原位置，卡组张数不变。

- 变化不会替换正在打出的那张卡（即触发变化的卡本身）
- 变化产生的卡牌来自当前流派池（2 个流派 + 通用）
- 可指定变化后的卡牌是否为强化状态
