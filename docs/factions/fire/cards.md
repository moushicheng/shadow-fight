# 火系 (FIRE) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：叠层引爆，一波秒杀

## 专属关键词：灼烧

- 灼烧层数 = 火系卡牌的额外伤害加成（如目标 6 层灼烧，10 伤害的火球实际造成 16）
- 部分卡牌可以「引爆」：消耗全部灼烧层数，造成 层数 × 倍率 的一次性爆发
- 灼烧不自然衰减，引爆后清零
- 非火系伤害不享受灼烧加成

## 设计思路

节奏是"蓄力 → 引爆"。前几张牌叠灼烧，后面引爆卡收割。卡序安排至关重要。

核心体验是"忍住不引爆 → 灼烧叠到够高 → 一张引爆卡带走对手"。

### 卡牌设计方向

1. **叠灼烧类** — 高效施加灼烧层数，为引爆做准备
2. **灼烧增伤类** — 普通火系伤害牌，享受灼烧加成
3. **引爆类** — 消耗全部灼烧层数，倍率转化为爆发伤害
4. **辅助蓄力类** — 提供生存/护甲，为蓄力争取时间

### 属性倾向

偏好高 STR（提高基础伤害）或高 SPD（更快循环到引爆卡）。

### 克制关系

- **克制** → 血族（引爆秒杀不给滚雪球时间）
- **被克** ← 咒术师（打乱卡序 = 打乱蓄力节奏）
