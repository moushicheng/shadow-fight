# 血族 (BLOOD) — 卡牌设计

## 派系定位

**关键词流派** | 赢法：偷属性滚雪球

## 专属关键词：汲取

- 「汲取」：从对方身上扣除属性并加到自己身上（如汲取 2 ATK = 对方 -2，己方 +2）
- 可汲取的属性：攻击力、速度、护甲
- 部分卡牌附带吸血效果（造成伤害回复 HP）
- 汲取效果持续本场战斗

## 设计思路

战斗越久越强、对手越弱，形成滚雪球。但前期偷取量不够时是弱势方，需要扛过开局。

核心体验是"每一次偷取都是双倍收益 → 差距越拉越大 → 后期碾压"。

### 卡牌设计方向

1. **汲取攻击类** — 偷取对手 ATK 的同时造成伤害
2. **汲取速度类** — 偷取对手 SPD，既加速自己又减速对手
3. **汲取护甲类** — 偷取对手护甲转为己用
4. **吸血续航类** — 造成伤害并回复 HP，提供前期生存
5. **滚雪球奖励类** — 根据已汲取总量获得额外效果

### 属性倾向

偏好高 CON（高血量扛过前期弱势期），高 SPD 也有利于更快完成汲取循环。

### 克制关系

- **克制** → 守卫/武僧（偷属性让精密体系崩溃）
- **被克** ← 火系（引爆秒杀不给滚雪球时间）
